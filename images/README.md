## Output Screenshots

This folder contains selected screenshots of key SQL outputs generated in PostgreSQL/pgAdmin.

Included outputs:
- `diagnosis_distribution.png` – counts of records by diagnosis category
- `demographics_gender_by_diagnosis.png` – gender distribution by diagnosis
- `lifestyle_smoke_drink_by_diagnosis.png` – smoking/drinking combinations by diagnosis
- `environment_factors_by_diagnosis.png` – environmental/infrastructure factors by diagnosis

These screenshots are included to demonstrate:
- query execution
- interpretable summary outputs
- portfolio-ready evidence of results
